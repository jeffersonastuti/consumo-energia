# ⚡ Calculadora de Consumo de Energia

Projeto desenvolvido em Python com o objetivo de estimar o consumo mensal de energia elétrica de aparelhos domésticos, com base em dados simples de uso diário.

---

## 📌 Funcionalidades

- Cálculo do consumo mensal em kWh
- Estimativa de custo mensal com base no valor do kWh
- Interface simples e intuitiva no terminal

---

## 🧮 Fórmula Utilizada

O consumo mensal é calculado com base na seguinte fórmula:

consumo = (potencia × horas_por_dia × 30) ÷ 1000

---

## 💡 Exemplo de Uso

Aparelho: Geladeira  
Consumo estimado: 45.00 kWh/mês  
Custo estimado: R$ 33.75 por mês  

---

## 🚀 Como Executar o Projeto

1. Clone o repositório:
git clone https://github.com/seu-usuario/consumo-energia.git

2. Acesse a pasta do projeto:
cd consumo-energia

3. Execute o programa:
python app.py
